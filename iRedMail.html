Project Detail

iRedMail adalah Open Source Full-Featured Mail Server yang secara otomatis menginstal dan mengkonfigurasi semua komponen mail server yang diperlukan di server Linux/BSD. Admin system dapat menghemat banyak waktu karena konfigurasi server yang kompleks sudah terinstall otomatis oleh iRedMail . Kotak surat dapat disimpan di MariaDB/MySQL, database PostgreSQL, atau OpenLDAP.
